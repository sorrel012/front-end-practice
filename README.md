# html-css 공부

## Visual Studio Code : https://code.visualstudio.com/download

### ✔단축키
- Tab: 들여쓰기

- Shift + Tab : 내어쓰기

- Ctrl + / : 주석, 주석 해제

- Ctrl + B : 사이드바 표시, 숨기기

- Ctrl + Shift + P : 명령어 팔레트

- Ctrl + Shift + D : 실행

### ✔기본 설정
- word wrap : 자동 줄바꿈

- font size : 글꼴 크기

### ✔확장 기능
- html preview
  - Ctrl + Shift + V : 실행 결과를 외부 웹 브라우저가 아닌 VS Code 내부에서 확인 가능
